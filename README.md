# News Feed


News Feed is a basic android application implementing:

  - RecyclerViews and ViewHolders
  - DrawerLayout
  - Navigation components
  - Safe args
  - Parcelable
  - ViewBinding
  - Retrofit and OkHttp
  - Network Error handling
  - Combining RxJava Observables
  - Callbacks with Generics
  - Glide
  - DateTime formating
