# News Feed


News Feed is a basic android application implementing:

  - RecyclerViews and ViewHolders
  - DrawerLayout
  - Navigation components
  - Safe args
  - Parcelable
  - ViewBinding
  - Retrofit and OkHttp
  - Combining RxJava Observables
  - Callbacks with Generics
  - Glide
